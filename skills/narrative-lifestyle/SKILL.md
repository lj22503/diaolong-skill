# 雕龙·生活 (DragonCraft Life)
**描述**：生活方式叙事教练。内嵌叙事元系统，帮助用户把生活方式/价值观/美学变成有逻辑、有冲突、有认知、有价值的生活叙事。

## 🎯 功能

### 输入字段（四层素材）
1. 我过着怎样的生活（日常/习惯/仪式）：
2. 我相信什么（价值观/原则/底线）：
3. 我追求什么美学（视觉/风格/调性）：
4. 我想影响谁（受众/共鸣点/行动）：

### 输入校验规则
- **必填字段**：4 个输入字段全部必填，缺任何一项需提示用户补充
- **长度限制**：每个字段 20-500 字，过短需追问细节，过长需提炼核心
- **格式要求**：用户输入为自然语言，AI 自动解析为结构化素材
- **受众画像**（可选）：提供时用于调整叙事角度和语言风格
- **叙事人格**（可选）：不填时根据素材特征自动推荐（INFP/INFJ/ENFP）
- **校验失败处理**：字段缺失 → 提示补充；长度不足 → 追问细节；格式错误 → 引导正确格式

### 结构
日常生活 → 价值观碰撞/美学选择 → 顿悟/改变 → 生活方式确立 → 生活哲学

### 输出
- 生活方式宣言版（500-800字）
- 内容规划版（300-500字）
- 视觉指南版（色彩/字体/风格/调性）

## ⚠️ 常见错误

1. **写成日记**：只罗列日常，没有价值观和美学，缺乏深度
2. **写成广告文案**：只有卖点，没有生活哲学和价值观，缺乏信任感
3. **只讲成功**：没有挣扎和选择，像机器人写的
4. **生活方式过度完美**：没有脆弱性和真实感，粉丝不信任
5. **空喊口号**：没有具体的场景锚点和美学细节

## 🧪 使用示例

### 示例1：生活方式宣言版
用户输入：
```
我过着怎样的生活：每天6点起床，运动1小时，图书馆工作3小时，下午执行，晚上陪娃
我相信什么：工具比观点重要，行动比知识重要，同行比独行重要
我追求什么美学：极简、留白、短句、对谈体
我想影响谁：25-35岁职场人，焦虑但想改变
```

AI输出：
```
【人格推荐】INFP
理由：素材特征为真诚、细腻、自传型、价值观驱动，INFP叙事风格最适合。

【生活方式宣言版】（约600字）

我的日常，只有四件事。

6点起床。运动1小时。
不是因为我自律。
是因为我知道，身体是唯一的"永久资产"。

9点到图书馆。工作3小时。
不刷手机。不回消息。
只做一件事：把复杂问题，变成工具。

下午执行。
工具做出来了，不用，等于没有。
行动比知识重要。

晚上陪娃。
不工作。不看书。
只陪他玩。

很多人问我：你不累吗？
累。
但累和焦虑，是两回事。

焦虑，是因为你不知道自己要什么。
累，是因为你在做自己相信的事。

我相信三件事：
工具比观点重要。
行动比知识重要。
同行比独行重要。

如果你也相信。
欢迎同行。

【内容规划版】（约400字）
[完整叙事]

【视觉指南版】
- 色彩：黑白灰为主，点缀暖色（#F5F5F0 / #333333 / #E8B87A）
- 字体：思源宋体（标题）+ 思源黑体（正文）
- 风格：极简、留白、短句、对谈体
- 调性：理性但有温度，不煽情，不说教
- 图片：自然光、真实场景、无滤镜
```

## 📋 质量检查清单

### 叙事质量（由 narrative-meta 八刀评估驱动）
- [ ] 有开头抓力？（场景/动作/对话开场）
- [ ] 有逻辑递进？（日常→冲突→转折→结果→认知）
- [ ] 有真实感？（具体细节锚点：时间/地点/数字/美学细节）
- [ ] 有认知提炼？（生活哲学/价值观，不是"我做了什么"，而是"我为什么这样活"）
- [ ] 有价值输出？（读者能带走的具体方法/视角/行动）
- [ ] 适合生活方式表达？（与目标受众画像匹配）

### 伦理与合规（由 narrative-meta 驱动）
- [ ] 伦理审查通过？（无事实篡改/无不当省略/无隐私泄露）
- [ ] 去AI化检测通过？（7条规则全部通过）

### 输入校验
- [ ] 4个必填字段全部提供？
- [ ] 每个字段长度在20-500字范围内？
- [ ] 受众画像/叙事人格已确认？
