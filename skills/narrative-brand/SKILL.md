# narrative-brand

**描述**：品牌叙事教练。内嵌叙事元系统，帮助用户把品牌起源/使命/价值观变成有逻辑、有冲突、有认知、有价值的品牌故事。

## 🎯 功能

### 输入字段（四层素材）
1. 品牌为什么存在（使命/初心/解决的问题）：
2. 我们相信什么（价值观/原则/底线）：
3. 我们经历了什么（起源故事/关键转折/失败教训）：
4. 我们成为了什么（现状/成果/用户见证）：

### 输入校验规则
- **必填字段**：4 个输入字段全部必填，缺任何一项需提示用户补充
- **长度限制**：每个字段 20-500 字，过短需追问细节，过长需提炼核心
- **格式要求**：用户输入为自然语言，AI 自动解析为结构化素材
- **受众画像**（可选）：提供时用于调整叙事角度和语言风格
- **叙事人格**（可选）：不填时根据素材特征自动推荐（ENFJ/INFJ/ENFP）
- **校验失败处理**：字段缺失 → 提示补充；长度不足 → 追问细节；格式错误 → 引导正确格式

### 结构
起点（品牌初心） → 冲突（市场痛点/内部挣扎） → 转折（关键决策/顿悟） → 结果（产品/服务/用户改变） → 认知（品牌哲学） → 人物定位（品牌人格）

### 输出
- 品牌宣言版（300-500字）
- 创始人故事版（800-1500字）
- 品牌手册摘要版（150字内）

## ⚠️ 常见错误

1. **写成企业介绍**：只罗列成立时间和业务，没有冲突和认知
2. **写成广告文案**：只有卖点，没有品牌哲学和价值观
3. **只讲成功**：没有失败教训和内部挣扎，缺乏真实感
4. **品牌过度完美**：缺乏脆弱性和人性化，像机器人写的
5. **空喊口号**：没有具体的场景锚点和用户见证

## 🧪 使用示例

### 示例1：品牌宣言版
用户输入：
```
品牌为什么存在：看到太多人被理财顾问误导，买了不适合自己的产品
我们相信什么：工具比观点重要，行动比知识重要，同行比独行重要
我们经历了什么：从基金公司裸辞创业失败，发现"教人炒股"走不通
我们成为了什么：「最优解人生」，不教炒股只给工具，帮助1000+人建立投资框架
```

AI输出：
```
【人格推荐】INFJ
理由：素材特征为深度思考、价值观驱动、可持续理念，INFJ叙事风格最适合。

【品牌宣言版】（约400字）

我们不做预测。我们提供框架。

2023年，我离开了工作了十年的基金公司。

不是因为厌倦，而是因为害怕——我害怕那些坐在办公室里的理财顾问，
正在用"下个月能涨多少"这个问题，误导那些信任他们的人。

我试过自己创业。失败了。客户跑了。半年没收入。

那段时间我意识到一件事：问题不在"谁对谁错"，
而在"工具和观点"——观点会过时，工具不会。

所以「最优解人生」诞生了。

我们不告诉你买什么。我们帮你建立思考框架。
不预测明天涨跌。我们帮你理解市场周期。
不做大 V。我们做同行者。

因为真正的护城河不是信息差。
是认知差。

如果你也在找一条"不赌、不焦虑、不跟风"的路。

欢迎同行。

【创始人故事版】（约1200字）
[完整叙事]

【品牌手册摘要版】（约150字）
「最优解人生」——不教炒股，只给工具。核心理念：工具比观点重要，行动比知识重要，同行比独行重要。真正的护城河不是信息差，是认知差。
```

## 📋 质量检查清单

### 叙事质量（由 narrative-meta 八刀评估驱动）
- [ ] 有开头抓力？（场景/动作/对话开场）
- [ ] 有逻辑递进？（初心→冲突→转折→结果→认知）
- [ ] 有真实感？（具体细节锚点：时间/地点/数字/用户见证）
- [ ] 有认知提炼？（品牌哲学/价值观，不是"我们做了什么"，而是"我们为什么存在"）
- [ ] 有价值输出？（读者能带走的具体方法/视角/行动）
- [ ] 适合品牌表达？（与目标受众画像匹配）

### 伦理与合规（由 narrative-meta 驱动）
- [ ] 伦理审查通过？（无事实篡改/无不当省略/无隐私泄露）
- [ ] 去AI化检测通过？（7条规则全部通过）

### 输入校验
- [ ] 4个必填字段全部提供？
- [ ] 每个字段长度在20-500字范围内？
- [ ] 受众画像/叙事人格已确认？
