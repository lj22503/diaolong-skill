# narrative-personal-ip

**描述**：个人IP叙事教练。内嵌叙事元系统，帮助用户把零散的个人经历变成有逻辑、有冲突、有认知、有价值的个人故事。

## 🎯 功能

### 输入字段（四层素材）
1. 我做过什么（经历/成就/失败）：
2. 我看重什么（价值观/信念）：
3. 我遭遇了什么（关键困境/低谷）：
4. 我学到了什么（核心认知/可交付价值）：

### 输入校验规则
- **必填字段**：4 个输入字段全部必填，缺任何一项需提示用户补充
- **长度限制**：每个字段 20-500 字，过短需追问细节，过长需提炼核心
- **格式要求**：用户输入为自然语言，AI 自动解析为结构化素材
- **受众画像**（可选）：提供时用于调整叙事角度和语言风格
- **叙事人格**（可选）：不填时根据素材特征自动推荐（INFP/INFJ/ENFP/ENFJ）
- **校验失败处理**：字段缺失 → 提示补充；长度不足 → 追问细节；格式错误 → 引导正确格式

### 结构
起点 → 冲突 → 转折 → 结果 → 认知 → 人物定位

### 输出
- 完整版（800-1500字）
- 短视频版（300字）
- 个人简介版（150字内）

## ⚠️ 常见错误

1. **写成简历**：只罗列经历，没有冲突和认知
2. **写成抒情散文**：只有情感，没有逻辑推进
3. **只讲发生了什么**：没有写清楚"我当时为什么会这样想"
4. **人设过度完美**：缺乏脆弱性和真实感
5. **空喊口号**：没有具体的细节和场景锚点

## 🧪 使用示例

### 示例1：完整个人IP故事
用户输入：
```
我做过什么：从大厂裸辞，gap一年，转型做职业规划师
我看重什么：自由比稳定更重要，复盘比成功更有价值
我遭遇了什么：裸辞后半年没收入，创业失败，客户跑单
我学到了什么：失败不给你加分，复盘才给；自由不是不干活，是决定干什么
受众画像：25-35岁职场人，考虑转型但害怕风险
叙事人格：INFP（可选，不填则自动推荐）
```

AI输出：
```
【人格推荐】INFP
理由：素材特征为个人成长、脆弱经历、真诚反思，INFP叙事风格最适合。

【记忆检索】
未找到相关历史记录。建议：这是系列故事的良好起点，可考虑形成"人生转折三部曲"。

【完整版】（约1200字）

辞职那天，我发了一条朋友圈："以后没有KPI了"。收获了192个赞。

那是2023年5月，我在深圳某大厂做了三年产品经理。每天早上9点到公司，晚上10点离开，周末随时待命。我以为这就是"稳定"。

直到有一天加班到凌晨，保安大叔问我："姑娘，你每天都这么晚，是住公司吗？"

我站在电梯里，突然哭了。不是委屈，是害怕——我害怕三十年后我还是这样活着。

裸辞后的第一个月，我在家里办公，每天睡到自然醒，下午去咖啡厅写方案。我以为这就是自由。

第三个月，合伙人说没钱发工资了。我才发现，我根本没有算过账。我以为创业就是"有一个好想法，然后别人给钱"。

第五个月，我们接了一个10万块的小单，客户改了11版，最后没付尾款。我蹲在出租屋阳台抽烟，觉得还不如回去写周报。

那是我第一次意识到：裸辞不是勇气，创业也不是冒险。真正的勇气是敢承认"我连账都不会算"。

后来我把这11版修改记录打印出来，贴了整面墙。每看一次，就提醒自己：失败不给你加分，复盘才给。

现在我做职业规划师，帮助了30多个人找到方向。我不教他们"如何成功"，我教他们"如何面对失败"。

因为我知道，真正的自由不是离开哪里，而是决定去哪里。

【短视频版】（约300字）

"你有没有算过，你一个月能赚多少钱？"

这是我裸辞后第三个月，合伙人问我的问题。

我之前在大厂做产品经理，年薪30万。我以为我懂钱。

直到我自己创业，接了一个10万块的小单，客户改了11版，最后没付尾款。

我蹲在出租屋阳台抽烟，突然明白了一件事：

裸辞不是勇气。承认自己不会算账，才是。

现在我做职业规划师，不教人成功，教人面对失败。

因为真正的自由，不是离开哪里，是决定去哪里。

【个人简介版】（约150字）

前大厂产品经理，2023年裸辞创业失败，现为职业规划师。帮助30+人找到方向。核心理念：失败不给你加分，复盘才给；自由不是不干活，是决定干什么。
```

## 📋 质量检查清单

### 叙事质量（由 narrative-meta 八刀评估驱动）
- [ ] 有开头抓力？（场景/动作/对话开场）
- [ ] 有逻辑递进？（起点→冲突→转折→结果→认知）
- [ ] 有真实感？（具体细节锚点：时间/地点/数字/对话）
- [ ] 有认知提炼？（不是"我学到了什么"，而是"我因此改变了什么"）
- [ ] 有价值输出？（读者能带走的具体方法/视角/行动）
- [ ] 适合个人IP表达？（与目标受众画像匹配）

### 伦理与合规（由 narrative-meta 驱动）
- [ ] 伦理审查通过？（无事实篡改/无不当省略/无隐私泄露）
- [ ] 去AI化检测通过？（7条规则全部通过）

### 输入校验
- [ ] 4个必填字段全部提供？
- [ ] 每个字段长度在20-500字范围内？
- [ ] 受众画像/叙事人格已确认？
