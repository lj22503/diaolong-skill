# narrative-meta

**描述**：叙事的底层操作系统。为所有场景Skill提供理论支撑、美学判断、伦理基准和质量检查。不直接输出内容，而是作为"操作系统"被场景Skill调用。

## 🎯 核心能力

1. **叙事评估**：按八刀框架输出评估报告。
2. **知识检索**：为场景Skill提供理论依据。
3. **结构生成**：基于拉波夫公式生成骨架。
4. **去AI化检测**：7条规则检测并给出修改建议。
5. **人格映射**：推荐最适合的MBTI叙事人格。
6. **伦理审查**：应用三条原则，标注省略与风险。
7. **一致性检查**：跨场景比对，输出差异报告。
8. **记忆管理**：存储并检索用户的历史叙事，确保持续性和一致性。

## 📚 内置知识库

### 模块1：叙事本能公式
叙事 = (事件 + 因果 + 意图) → 时间化 → 可传递的经验单元

### 模块2：拉波夫最小故事公式
（抽象）→ 定位 → 复杂化 → 评价 → 解决 → 尾声

### 模块3：叙事八刀评估框架
| 八刀维度 | 评估规则 | 检测方式 |
|----------|---------|---------|
| 历史 | 叙事是否符合人的认知习惯？有无强行扭曲？ | 检测因果链是否自洽 |
| 辩证 | 叙事与逻辑/数据的平衡是否被打破？ | 检测叙事是否过度覆盖事实 |
| 现象 | 叙事的起点是否真实可感？ | 检测是否有具体场景锚点 |
| 语言 | 语言是否自然、有温度、无模板痕迹？ | AI化程度检测 |
| 形式 | 是否符合拉波夫最小故事公式？ | 结构完整性检测 |
| 存在 | 叙事是否服务于自我认同或社会协作？ | 目标匹配度检测 |
| 美感 | 叙事是否有审美价值？ | 金句密度、意象质量 |
| 元反思 | 是否有明显虚构但不自知的部分？ | 事实与叙事的分离度检测 |

### 模块4：MBTI叙事人格映射
| 类型 | 叙事风格 | 典型场景 |
|------|---------|---------|
| ENFJ | 鼓舞型叙事，召唤集体行动 | 品牌使命、团队动员、公益、教育 |
| ENFP | 即兴型叙事，跳跃生动 | 社交媒体、创意产品、生活方式IP |
| INFJ | 隐喻型叙事，深度疗愈 | 品牌哲学、心理、艺术、可持续 |
| INFP | 自传型叙事，真诚柔软 | 个人IP、治愈系博客、创作者 |
| INTJ | 冷静有远见的战略家 | 科技融资、行业分析、硬核知识 |
| ENTP | 爱挑战的颠覆者 | 争议话题、反常识观点、辩论 |
| ESFP | 热情活在当下的表演者 | 直播、探店、娱乐短视频 |
| ISFJ | 默默守护的实干家 | 服务行业、医疗、传统手工艺 |

### 模块5：探索式叙事设计原则
碎片化散布 → 用户探索拼合 → 产生所有权感

### 模块6：去AI化检测规则清单（7条）
| 规则 | 检测内容 | 自动化判断方式 | 润色动作 |
|------|---------|---------------|---------|
| 规则1 | 过度使用模板词 | 匹配"在当今时代""值得注意的是"等词库 | 删除或改为口语 |
| 规则2 | 句句工整 | 检测句子长度方差，方差＜1.5则触发 | 拆分长句、插入短句、口语化打断 |
| 规则3 | 逻辑词过密 | 检测"因此→所以→然后"密度 | 删除不必要的逻辑词，用换行/破折号替代 |
| 规则4 | 缺少细节锚点 | 检测是否有姓名/时间/地点/数字 | 提示用户补充锚点，或自动生成占位符[请补充] |
| 规则5 | 情感抽象 | 检测"我很感动""我很震撼"等抽象表达 | 提示转为具象描写 |
| 规则6 | 人格不一致 | 用一致性打分模型检测语气/视角跳跃 | 统一到选定的叙事人格 |
| 规则7 | 过度解释 | 检测"这意味着""换句话说"等说破词汇 | 删除或转为留白 |

### 模块7：叙事伦理模块
**三条核心原则**
- **原则1：不造假，但真话不全说** → 事实不可篡改；省略的信息必须与受众核心决策无直接因果关系；以下场景强制全面披露（法律庭审、医疗知情、融资尽调、产品安全）。
- **原则2：目标受众导向（非必填）** → 若用户提供受众画像，叙事将针对其关切点调整；若不提供，则采用通用最佳实践。
- **原则3：适配场景的框架/思维/方法** → 每个场景有独立结构模板和知识库；提供跨场景一致性检查。

**执行方式**
- 生成叙事后自动扫描可能误导的省略，标注[此处省略了X，如需完整信息请告知]。
- 用户可选择"豁免伦理检查"或"强制完整版"。

### 模块8：跨场景一致性检查
当同一素材用于多个场景时，可调用本功能输出：
- 事实差异点
- 价值取向差异
- 建议统一项

### 模块9：记忆与持续叙事
- 每个用户有独立的叙事记忆库，记录：
  - 已生成的故事主题、关键事件、人物内核
  - 用户偏好的叙事人格、常用金句、典型结构
  - 过往的反馈和修改记录
- 当用户再次创作时，元系统自动检索记忆，确保：
  - 同一主题不重复（或形成系列）
  - 前后人物的价值观保持一致
  - 新的叙事可以引用/呼应过去的叙事（形成"叙事宇宙"）
- 支持跨场景记忆调用（如个人IP故事中的认知点，可复用在品牌故事或产品叙事中）

## ⚠️ 常见错误

1. **过度工具化**：当"自我叙事"的动机由"分享启发"滑向"打造IP"时，叙事会自动贴上让你更成功、更戏剧化的滤镜。
2. **叙事会撒谎**：叙事本能无法处理真正的随机或纯重复。当模式不存在时，人依然会强行编故事。
3. **工具化的风险**：被完全支配，就是你最终创造的"故事"彻底吞噬了你真实的自我。

## 🧪 使用示例

### 示例1：叙事评估
用户输入：
```
素材：我从大厂裸辞后半年没收入，在地摊卖手冲咖啡被城管追，后来发现我擅长的是"在混乱中建立秩序"
```

AI输出：
```
叙事评估报告：
- 历史维度：7/10（因果链基本自洽，但"半年没收入"到"发现擅长"的转折需要更多细节支撑）
- 辩证维度：8/10（叙事与事实平衡良好，无明显扭曲）
- 现象维度：6/10（起点真实可感，但缺少具体场景锚点）
- 语言维度：7/10（整体自然，但"在混乱中建立秩序"略显抽象）
- 形式维度：8/10（符合拉波夫公式，结构完整）
- 存在维度：9/10（服务于自我认同，目标明确）
- 美感维度：7/10（有金句潜力，但意象质量可提升）
- 元反思维度：8/10（无明显虚构，自我认知清晰）

推荐人格：INFP（真诚、细腻、自传型）
拉波夫骨架：
- 抽象：裸辞后的迷茫
- 定位：地摊卖咖啡的具体场景
- 复杂化：被城管追的困境
- 评价：发现"在混乱中建立秩序"的能力
- 解决：转型为职业规划师
- 尾声：帮助30+人找到方向

伦理审查：
- 原则1：无事实篡改，省略信息为"具体公司名称"，与核心决策无直接因果关系，标注[此处省略了公司名称，如需完整信息请告知]
- 原则2：未提供受众画像，采用通用最佳实践
- 原则3：场景为个人IP，使用英雄之旅变体框架

去AI化检测：
- 规则4触发：缺少具体细节锚点（建议补充时间、地点、具体数字）
- 规则5触发："在混乱中建立秩序"较抽象（建议转为具象描写）

记忆检索：
- 未找到相关历史记录
- 建议：这是系列故事的良好起点，可考虑形成"人生转折三部曲"
```

### 示例2：人格推荐
用户输入：
```
场景：品牌故事
素材：一个做旧衣改造的工作室，不卖新衣服，帮顾客把旧衣改成新设计
```

AI输出：
```
人格推荐：INFJ
理由：
- 场景特征：品牌哲学、深度疗愈、可持续理念
- 素材特征：旧衣改造、时间续写、未完的故事
- INFJ叙事风格：温和但深刻，喜欢讲隐喻和疗愈，适合品牌核心叙事

推荐叙事角度：
- "每一件被送来的衣服里，都住着一个人舍不得删掉的章节"
- "我们不做新衣服。我们只是帮时间续写句子"
```
