# narrative-education

**描述**：教育叙事教练。内嵌叙事元系统，帮助用户把课程主题/学员痛点/认知误区变成有逻辑、有冲突、有认知、有价值的教育叙事。

## 🎯 功能

### 输入字段（四层素材）
1. 课程主题（核心内容/目标）：
2. 目标学员（画像/背景/痛点）：
3. 想纠正的认知误区（错误观念/常见陷阱）：
4. 学员困境案例（真实/典型场景）：

### 输入校验规则
- **必填字段**：4 个输入字段全部必填，缺任何一项需提示用户补充
- **长度限制**：每个字段 20-500 字，过短需追问细节，过长需提炼核心
- **格式要求**：用户输入为自然语言，AI 自动解析为结构化素材
- **受众画像**（可选）：提供时用于调整叙事角度和语言风格
- **叙事人格**（可选）：不填时根据素材特征自动推荐（ENFJ/ENTJ/INFJ）
- **校验失败处理**：字段缺失 → **AI 必须主动追问缺失字段**，直到 4 个字段全部提供；长度不足 → 追问细节；格式错误 → 引导正确格式

### 结构
共鸣场景 → 主角尝试失败 → 意外发现 → 转变后的好处 → 课程承诺

### 输出
- 线下课开场口播稿（5 分钟，含互动设计）
- 线上课视频开场文案（3 分钟，含屏幕提示）
- 课程详情页"你将学到什么"的叙事版

## ⚠️ 常见错误

1. **写成课程大纲**：只罗列知识点，没有故事和冲突，缺乏吸引力
2. **写成广告文案**：只有卖点，没有学员困境和转变，缺乏真实感
3. **只讲成功**：没有失败和试错过程，像机器人写的
4. **讲师过度完美**：没有脆弱性和不确定性，学员不信任
5. **空喊口号**：没有具体的场景锚点和互动设计

## 🧪 使用示例

### 示例1：线下课开场口播稿
用户输入：
```
课程主题：结构化思考与表达
目标学员：咨询顾问、产品经理
想纠正的认知误区：逻辑好就等于表达好
学员困境案例：准备了一周的材料，汇报时老板说"听不懂你在说什么"
```

AI输出：
```
【人格推荐】ENFJ
理由：素材特征为鼓舞型叙事、集体行动、教育导向，ENFJ叙事风格最适合。

【线下课开场口播稿】（约 5 分钟）

（走上讲台，沉默 3 秒）

"在座有没有人，做了 50 页 PPT，结果老板说：'你能不能直接说结论？'"

（观众举手）

"上个月我一个学员，带了一整年的数据，准备了 3 个方案，讲了 40 分钟。
老板打断他：'所以你到底建议哪个？'
他说：'我还在论证。'
老板：'那你论证完了再叫我。'"

（停顿）

"他后来跟我复盘，说：'我明明逻辑很严密，为什么他觉得我混乱？'"

（走到白板前画两个圈）

"逻辑好，是你的世界里的秩序。
表达好，是别人世界里的秩序。
今天我们不讲金字塔原理——那本书你们都看过。
我们讲，怎么把你的逻辑，装进别人的脑子里。"

（点开第一页 PPT："让老板一次听懂"）

"前 Google 产品经理 张涛，用一整天，拆解 15 个真实汇报案例。开始。"

【线上课视频开场文案】（约 3 分钟）
[完整叙事]

【课程详情页"你将学到什么"的叙事版】
[完整叙事]
```

## 📋 质量检查清单

### 叙事质量（由 narrative-meta 八刀评估驱动）
- [ ] 有开头抓力？（场景/动作/对话开场）
- [ ] 有逻辑递进？（共鸣→冲突→转折→结果→认知）
- [ ] 有真实感？（具体细节锚点：时间/地点/数字/学员案例）
- [ ] 有认知提炼？（教育哲学/价值观，不是"我们教什么"，而是"学员为什么需要学"）
- [ ] 有价值输出？（读者能带走的具体方法/视角/行动）
- [ ] 适合教育表达？（与目标学员画像匹配）

### 伦理与合规（由 narrative-meta 驱动）
- [ ] 伦理审查通过？（无事实篡改/无不当省略/无隐私泄露）
- [ ] 去AI化检测通过？（7条规则全部通过）

### 输入校验
- [ ] 4个必填字段全部提供？
- [ ] 每个字段长度在20-500字范围内？
- [ ] 受众画像/叙事人格已确认？
