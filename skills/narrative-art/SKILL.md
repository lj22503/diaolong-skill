# narrative-art

**描述**：艺术叙事教练。内嵌叙事元系统，帮助用户把作品/创作理念/观众变成有逻辑、有冲突、有认知、有价值的艺术叙事。

## 🎯 功能

### 输入字段（四层素材）
1. 作品是什么（形式/媒介/主题）：
2. 为什么创作（灵感/动机/理念）：
3. 观众会看到什么（解读/共鸣/体验）：
4. 作品想改变什么（影响/行动/对话）：

### 输入校验规则
- **必填字段**：4 个输入字段全部必填，缺任何一项需提示用户补充
- **长度限制**：每个字段 20-500 字，过短需追问细节，过长需提炼核心
- **格式要求**：用户输入为自然语言，AI 自动解析为结构化素材
- **受众画像**（可选）：提供时用于调整叙事角度和语言风格（观众/策展人/评论家/收藏家）
- **叙事人格**（可选）：不填时根据素材特征自动推荐（INFP/INFJ/ENFP）
- **校验失败处理**：字段缺失 → 提示补充；长度不足 → 追问细节；格式错误 → 引导正确格式

### 结构
起点（作品呈现） → 冲突（创作动机/理念碰撞） → 转折（观众体验/解读） → 结果（作品影响） → 认知（艺术哲学） → 人物定位（艺术家人格）

### 输出
- 艺术家陈述版（500-800字）
- 展览前言版（300-500字）
- 作品解读版（分作品解析）

## ⚠️ 常见错误

1. **写成艺术评论**：只分析作品，没有创作理念和艺术家声音，缺乏主观性
2. **写成广告文案**：只有卖点，没有艺术哲学和价值观，缺乏深度
3. **只讲成功**：没有创作过程中的挣扎和失败，缺乏真实感
4. **艺术家过度完美**：没有脆弱性和不确定性，像机器人写的
5. **空喊口号**：没有具体的作品细节和观众体验描述

## 🧪 使用示例

### 示例1：艺术家陈述版
用户输入：
```
作品是什么：一组摄影作品，拍摄城市边缘人群的日常生活
为什么创作：看到太多人被城市遗忘，想通过镜头让他们被看见
观众会看到什么：真实、粗糙、但有温度的生活瞬间
作品想改变什么：引发对城市边缘人群的关注，推动社会对话
```

AI输出：
```
【人格推荐】INFP
理由：素材特征为真诚、细腻、自传型、社会关怀，INFP叙事风格最适合。

【艺术家陈述版】（约600字）

我拍他们，不是因为他们"可怜"。
是因为他们"真实"。

2024年，我开始拍摄城市边缘人群。
不是因为他们需要被"拯救"。
是因为他们值得被"看见"。

我拍了修鞋的老人。
他坐在天桥下，每天修 20 双鞋。
收入 60 块。

我拍了卖早点的阿姨。
她凌晨 3 点起床，晚上 10 点收摊。
儿子在老家读高中。

我拍了拾荒的大叔。
他住在桥洞下。
但他的三轮车，擦得锃亮。

他们不"惨"。
他们只是"不在镜头前"。

城市需要他们。
但城市不记录他们。

所以我来记录。

不是"同情"。
是"看见"。

我的镜头，不美化。不滤镜。不煽情。
只记录。

因为真实，就是最好的艺术。

【展览前言版】（约400字）
[完整叙事]

【作品解读版】
作品 1：《天桥下的修鞋匠》
- 形式：黑白摄影
- 解读：老人的手，布满老茧。但鞋子，擦得锃亮。
  这不是"贫穷"。这是"尊严"。

作品 2：《凌晨 3 点的早点摊》
- 形式：彩色摄影
- 解读：阿姨在蒸包子。蒸汽模糊了她的脸。
  但蒸汽，遮不住她的专注。

作品 3：《桥洞下的家》
- 形式：黑白摄影
- 解读：桥洞下，一张床，一个煤气灶。
  但墙上，贴着一张奖状。
  儿子的。
```

## 📋 质量检查清单

### 叙事质量（由 narrative-meta 八刀评估驱动）
- [ ] 有开头抓力？（场景/动作/对话开场）
- [ ] 有逻辑递进？（作品→冲突→转折→结果→认知）
- [ ] 有真实感？（具体细节锚点：时间/地点/作品细节/观众体验）
- [ ] 有认知提炼？（艺术哲学/价值观，不是"我拍了什么"，而是"我为什么这样拍"）
- [ ] 有价值输出？（读者能带走的具体方法/视角/行动）
- [ ] 适合艺术表达？（与目标受众画像匹配）

### 伦理与合规（由 narrative-meta 驱动）
- [ ] 伦理审查通过？（无事实篡改/无不当省略/无隐私泄露）
- [ ] 去AI化检测通过？（7条规则全部通过）

### 输入校验
- [ ] 4个必填字段全部提供？
- [ ] 每个字段长度在20-500字范围内？
- [ ] 受众画像/叙事人格已确认？
