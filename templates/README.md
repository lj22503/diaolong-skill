## 📖 叙事价值观

我们选择的案例不是为了展示"如何成功"，而是为了展示"如何用叙事让世界多一点点善意"。

当下人们需要的故事不再是"逆袭神话"，而是：

- 🤝 **我看见了你的不容易**（共情）
- 🌱 **即使很小，我也愿意做**（行动）
- 🔗 **我们不完美，但在一起**（连接）
- ✨ **伤口可以变成光**（希望）

## 📊 使用说明

### 如何使用模板

1. **选择场景**：根据你的需求选择对应的场景文件
2. **阅读案例**：每个场景有4个真实案例改编
3. **提取框架**：每个案例标注了"发于微小"和"放大善意"的叙事结构
4. **替换素材**：用你自己的素材替换案例中的细节
5. **调用叙事skill**：用 narrative-workflow 生成打磨后的版本

### 模板结构

每个案例包含：
- **原始素材**：真实案例简介
- **打磨后故事**：300-500字完整版
- **核心金句**：记忆点
- **叙事结构**：发于微小 → 放大善意
