### 回答格式

```

主体：3-5个观点，每个观点有案例支撑
结尾：引导关注/CTA
```

### 示例

**问题**：如何写出打动人心的个人故事？

**回答**：

```
好故事的公式不是"逆袭"，而是"发于微小，放大善意"。

什么意思？

举个例子。

一个退役警犬被辅警领养的故事。

差的写法：
"一位辅警领养了退役警犬，体现了忠诚的双向奔赴。"

好的写法：
"它叫黑豹。不是因为它黑。是因为它年轻时跑得像豹子。
它在警队工作了8年。搜过毒，搜过爆，立过二等功。
去年，它退役了。
警队说：'可以领养。'
我举手了。
同事说：'它老了，你养它干嘛？'
我说：'它老了，我养它。'"

区别在哪？

1. 有具体细节（8年、二等功、8块瓦）
2. 有场景锚点（警队、举手、同事对话）
3. 有金句（"它老了，我养它"）
4. 有情感（不是煽情，是克制）

这就是叙事框架的力量。

不是你不会写。是你没有框架。

我整理了15个场景的叙事模板，60个真实案例改编。
需要的话，关注我，回复'模板'获取。
```
